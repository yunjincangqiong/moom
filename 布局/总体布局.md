## 双飞翼(两边固定中间自适应)

### 定位

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    * {
      padding: 0;
      margin: 0;
    }

    html, body{
      height: 100%;
    }
	/* 父元素设置相对定位 */
    div.container {
      /* 方法1: 设置父元素的左右内边距为固定盒子宽度 */
      padding: 0 100px;
      height: 100%;
      position: relative;
    } 
	/* 两边使用绝对定位, 将盒子定位在左右两边 */
    div.left {
      position: absolute;
      left: 0;
      top: 0;
      width: 100px;
      height: 100%;
      background-color: red;
    }

    div.right {
      position: absolute;
      right: 0;
      top: 0;
      width: 100px;
      height: 100%;
      background-color: blue;
    }
	/* 宽度不设置默认自适应 */
    div.center {
      /* 方法2: 自适应元素两边空出来固定盒子的宽度 */
      margin-left: 100px;
      margin-right: 100px;
      height: 100%;
      background-color: yellow; 
    }
  </style>
</head>

<body>
  <div class="container">
    <div class="left">1</div>
    <div class="center">2</div>
    <div class="right">3</div>
  </div>
</body>

</html>
```

### flex

```css
.parent {
  display: flex;
}

.left {
  width: 100px;
}

.right {
  width: 100px;
}

.center {
  flex: 1;
}
```



## 横向一边固定另一边自适应

### 浮动 + overflow: hidden

```css
.parent {
  height: 100%;
}
.child1 {
  /* 左边固定 */
  float: left;
  /* 右边固定 */
  /* float: right; */
  width: 100px;
  height: 100%;
}
.child2 {
  /*让这个元素绝对绝缘  触发bfc*/
  /*不于其他浮动元素有交集*/
  overflow: hidden;
  height: 100%;
}
```

### 浮动 + margin-left/margin-right

```css
.parent {
  height: 100%;
}
.child1 {
  /* 左边固定 */
  float: left;
  /* 右边固定 */
  /* float: right; */
  width: 100px;
}
.child2 {
  /* 用margin让出child1宽度 */
  margin-left: 100px;
  /* margin-right: 100px; */
  height: 100%;
}
```

## 实现固宽高比的盒子

原理: 借用 padding 百分之值相对于父元素的宽进行设置

下面以 16 : 9 为例

```html
<style>
  .parent {
    width: 50%;
    /* 父元素相对定位 */
    position: relative;
  }

  /* 作用: 将父元素撑起来使其达到固定宽高比 */
  .son1 {
    /* 计算比例的公式为 100% * 宽 / 高 */
    /* 100% * 9 / 16 = 56.25% */
    /* 第一个子元素设置相应比例的 padding-top */
    /* 也可以设置 margin-top, 但是为了防止元素塌陷, 不建议使用 */
    padding-top: 56.25%;
  }

  /* 内容写在这个盒子中 */
  .son2 {
    /* 以下属性缺一不可 */
    /* 相对定位 */
    position: absolute;
    /* 设置默认值 */
    top: 0;
    left: 0;
    /* 使用父元素的宽高 */
    height: 100%;
    width: 100%;
    
    /* 第2种方法 */
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
  }
</style>

<body>
  <div class="parent">
    <div class="son1"></div>
    <div class="son2">..内容..</div>
  </div>
</body>
```
