## 布局说明

### position

```
元素设置 position 属性时, 只要不为默认值, 自身就会变成块级元素
绝对定位和固定定位一般要设置 left 和 top 值, 不设置时绝对定位和固定定位的元素默认就固定在原始文档流的位置处
```

### display

```
元素设置 flex 属性值时, 会影响其内部的子元素的 display 值, 会使其变成 flex 盒
```

### float

```
元素设置 float 属性时, 只要不为默认值, 自身就变为块级元素
```

## 元素垂直居中

### 定位

```css
.parent {
	position: relative;
}

.child {
  	position: absolute;
    /* 一定要设值margin, 否则子元素就变成了宽高与父元素一致, 而不是垂直居中 */
    margin: auto;
  	top: 0;
  	left: 0;
  	bottom: 0;
  	right: 0;
}
```

### 定位 + margin

```css
原理: 子绝父相, left和top属性值相对于父元素的宽高, margin-top/left 值设置为自身宽高的一半.
子元素 通用 :

.parent {
  position: relative;
}

.child {
  position: absolute;
  left: 50%;
  top: 50%;
  width: 50px;
  height: 50px;
  margin-left: -25px;
  margin-top: -25px;
}
```

### 定位 + transform

```css
原理: 子绝父相, left和top属性值相对于父元素的宽高, translate属性值相对于自身的宽高.
子元素 通用 :

.parent {
  position: relative;
}

.child {
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}
```

### table-cell

```css
原理: 单元格盒子独有的垂直居中特性, 块级元素margin: 0 auto;左右居中, 行内/行内块元素父元素设置text-algin: center;左右居中.
子元素 为块级元素 :

.parent {
  display: table-cell;
  vertical-align: middle;
}

.child {
  margin: 0 auto;
}

子元素为 行内/行内块元素 :

.parent {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
```

### flex

```html
原理: 弹性盒模型的 主轴 和 交叉轴 居中设置
子元素 通用 :
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    .wrapper {
      width: 100px;
      height: 100px;
      display: flex;
      align-items: center;
      justify-content: center;
      background-color: red;
    }

    .container {
      width: 50px;
      height: 50px;
      background-color: #123;
    }
  </style>
</head>
<body>
  <div class="wrapper">
    <div class="container">asd</div>
  </div>
</body>
</html>
```

## 塌陷问题

```html
产生原因: 父元素外部没有限制 
问题: 子元素设置 margin-top 值时自身不会相对于父元素偏移, 而是会导致包括父元素甚至祖先元素在内的整体移动
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    .container {
      width: 100px;
      height: 100px;
      background-color: #123;
      /* 第一种解决方案: 设置 border-top */
      /* border-top: 1px solid transparent; */
      /* 第二种解决方案: 设置 padding-top */
      /* padding-top: 1px; */
      /* 第三种解决方案: 设置 overflow 不为默认值即可, 一般为 hidden */
      /* overflow: auto; */
      /* 第四种解决方案: 让父元素浮动 */
      float: left;
    }
	
    /* 第五种解决方案: 添加伪元素 */
    /* !!注意此方案为固定写法!! */
    /* .container::before {
      content: "";
      display: table;
    } */

    .wrapper {
      width: 50px;
      height: 50px;
      background-color: #789;
      margin-top: 30px;
    }
  </style>
</head>
<body>
  <div class="container">
    <div class="wrapper"></div>
  </div>
</body>
</html>
```

## 清除浮动

### 父元素定义 height

### 添加最后一个子元素空块级元素(一般为div)

```css
css: 
	div {
      width: 0;
      height: 0;
      clear: both;
    }
```

### 父元素::after + zoom(兼容ie)(最常用)

```css
.clearfix::after {
  content: '';
  display: block;
  clear: both;
  visibility:hidden;
  height:0
}
.clearfix {
  zoom: 1
}
```
